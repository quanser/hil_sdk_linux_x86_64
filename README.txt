HIL SDK 2019 SP1 (3.8.2866)
-------------------------------

Thank you for using HIL SDK 2019 SP1!

We look forward to supporting your real-time and embedded development needs in the future. 

To install HIL SDK on Linux x86_64:
- Copy all the files in the desired target directory onto the target.
- On the target computer, uninstall previous versions of HIL SDK on the target by running the following command:
    sudo uninstall_hil_sdk
- On the target computer, go to the directory where you copied the files and run the following commands:
    chmod a+x setup_hil_sdk uninstall_hil_sdk
    sudo ./setup_hil_sdk
- Type "yes" (without quotes) in reponse to the license agreement prompt if you agree with the license.

HIL SDK documentation and examples on Linux x86_64:
- Documentation can be found under /opt/quanser/hil_sdk/help/index.html
- Examples can be found under /opt/quanser/hil_sdk/examples
- Note that the executables generated by HIL SDK require root access, therefore all executables should be called with "sudo". For example the "hil_read_analog_example" executable should be invoked as:
    sudo ./hil_read_analog_example

To uninstall HIL SDK on Linux Linux x86_64:
- Run the following command:
    sudo uninstall_hil_sdk
